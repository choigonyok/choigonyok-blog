가독성 좋은 코드란?

>에러처리
```
err := nil {log.fatalln(err)}
```

보다는

```
err := nil {
        log.fatalln(err)
}
```

<br>

> 변수이름
```
var a, aa, aaa int
```

보다는

```
var project_num, review_num, study_num int
```

<br>

> 쿼리문 작성
```
c.Query("SELECT * FROM table")
```

보다는

```
query := "SELECT * FROM table"
c.Query(query)
```
